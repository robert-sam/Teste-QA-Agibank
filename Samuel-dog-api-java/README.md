## QA Teste Samuel – Dog API

# Tecnologias

* Java
* RestAssured
* JUnit 5
* Maven

# Execução

mvn clean test

# Cenários

* Listagem de raças
* Imagens por raça
* Imagem aleatória

# Estratégia

Os testes validam:

* Status HTTP
* Estrutura do JSON
* Conteúdo retornado

Foco em garantir a integridade e confiabilidade da API.
